Ability Tick Count

Returns the ticks the ability in the context has been enabled for.

Settings:

Setting Type Description Required Fallback Value
modify MoLang MoLang function to modify the value. false /
ability Ability Reference The ability to get the ticks from. Defined in this syntax: # true /

Example:

{"type":"palladium:ability_tick_count","ability":"namespace:example_power#ability_key"}

Condition

Accepts to condition(s) to use different strings as variables

Settings:

Setting Type Description Required Fallback Value
condition Condition | Condition[] The condition(s) to check. Can be a single condition or a list of conditions. true /
on_false string The value to return if the result is false. true /
on_true string The value to return if the result is true. true /

Example:

{"type":"palladium:condition","condition":{"type":"palladium:crouching"},"on_true":"is_crouching","on_false":"not_crouching"}
