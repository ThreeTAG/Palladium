Texture Transformers

Overview

Palladium

  • Alpha Mask
  • Overlay

Alpha Mask

Applies an alpha mask to the texture.

Settings:

Setting Type Description Required Fallback Value
mask ResourceLocation The location of the mask texture. Can use variables true /

Example:

{"type":"palladium:alpha_mask","mask":"example:mask"}

Overlay

Puts another texture on top of the base texture.

Settings:

Setting Type Description Required Fallback Value
overlay ResourceLocation The location of the overlay texture. Can use variables true /
ignore_blank boolean If true, the overlay will not be applied to blank pixels. false false

Example:

{"type":"palladium:overlay","overlay":"example:mask"}
