Geo Render Layer

Uses GeckoLib to render a geo model, including animations

Settings:

Setting Type Description Required Fallback Value
light_emission integer The light emission of the model. Must be within 0 - 15 false 0
texture Texture Reference The texture of the model true /
bones Map The bones of the model. You can specify here what the bones for each body part are called in your model file. false {HEAD=head, BODY=body, RIGHT_ARM=right_arm, LEFT_ARM=left_arm, RIGHT_LEG=right_leg, LEFT_LEG=left_leg}
animations ResourceLocation The animations to use false /
model ResourceLocation The model to render true /
animation_controller Animation Controller | Animation Controller[] The animation controllers to control or trigger the animations. false /

Compound Render Layer

A compound render layer that can contain multiple render layers.

Settings:

Setting Type Description Required Fallback Value
layers Render Layers The list of render layers this compound render layer contains. true /

Default Render Layer

Default render layer that renders a model with a texture.

Settings:

Setting Type Description Required Fallback Value
light_emission integer The light emission of the model. Must be within 0 - 15 false 0
texture Texture Reference The texture to render the model with. true /
model_layer ResourceLocation The model layer to render. false If not present, the model of the parent entity will be used.
render_type "minecraft:solid" | "minecraft:cutout" The render type to render the model with. false minecraft:solid

Example:

{"type":"palladium:render_layer","light_emission":5,"model_layer":{"wide":"example:wide_model#main","slim":"example:slim_model#main"},"texture":{"wide":"example:wide_texture","slim":"example:slim_texture"}}

Lightning Sparks

Renders lightning sparks around the entity.

Settings:

Setting Type Description Required Fallback Value
amount integer The amount of sparks to spawn during each occurence. false 5
render_settings Laser Renderer Settings The render settings for the sparks. true /
frequency float The frequency of the sparks. false 0.5

Example:

{"type":"palladium:lightning_sparks","frequency":0.2,"amount":10,"render_settings":{"size":2.0,"glow":"#0000ff","bloom":2}}
